# IP TV Player

Современный Windows десктоп-плеер для просмотра IPTV потоков на основе C#/.NET 6 с WPF и LibVLCSharp.

## Возможности

### Плейлисты и каналы
- Импорт IPTV плейлистов из файла (.m3u, .m3u8) и по URL
- Drag & drop M3U-файлов прямо в окно
- Парсинг и отображение каналов с логотипами и группировкой
- Поиск каналов по названию, фильтр по группам
- Управление несколькими плейлистами (удаление, переименование, обновление)
- ★ Избранные каналы — переключение звёздочкой, отдельный фильтр
- 🕒 История просмотра — последние просмотренные каналы

### Плеер
- Встроенный видеоплеер LibVLCSharp с поддержкой HLS/IPTV потоков
- Полноэкранный режим (F / F11 / двойной клик)
- Скриншот текущего эфира (📸)
- Запись потока в файл (⏺ → .ts)
- Переключение аудиодорожек и субтитров (🎵)
- Отключение/включение звука (M)
- Регулировка громкости (↑/↓, ползунок)
- Индикатор буферизации
- Информация о потоке (разрешение, FPS, аудиоканалы) в заголовке

### EPG (программа передач)
- Парсинг XMLTV формата
- Кэширование с TTL 6 часов
- Текущая программа в заголовке окна
- 📅 Окно расписания на 24 часа для выбранного канала
- Привязка по TVG-ID канала

### UI / UX
- Тёмная минималистичная тема
- Контекстные меню на плейлистах и каналах
- Восстановление позиции и размера окна при запуске
- Сохранение настроек (громкость, путь скриншотов, EPG URL)

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| **Space** | Воспроизведение / пауза |
| **S** | Остановить |
| **F / F11** | Полноэкранный режим |
| **Esc** | Выйти из полного экрана |
| **M** | Mute / unmute |
| **↑ / ↓** | Громкость ±5 |
| **← / →** | Предыдущий / следующий канал |
| **F1** | Справка по хоткеям |

## Требования

- Windows 7 SP1 или выше
- .NET 6.0 Runtime
- LibVLC (поставляется через NuGet)

## Технологический стек

- **Framework**: .NET 6.0 (net6.0-windows)
- **UI**: WPF
- **Видеоплеер**: LibVLCSharp 3.6.8
- **База данных**: SQLite (System.Data.SQLite)
- **Логирование**: Serilog (Sinks.File)
- **Парсинг**: System.Xml.Linq (XMLTV), кастомный M3U

## База данных

Таблицы SQLite:
- `Playlists` — импортированные плейлисты
- `Channels` — каналы плейлиста, флаг `IsFavorite`
- `WatchHistory` — записи о просмотрах
- `EpgPrograms` — программа передач (индекс по TVG-ID и времени)
- `EpgSource` — TTL-маркер для источника EPG

## Решение проблем

### Видео не воспроизводится
- Проверьте, что в `bin/Debug/net6.0-windows/libvlc/` есть DLL
- Убедитесь, что URL потока корректен
- Логи в `Logs/app-*.txt`

### Не загружается плейлист
- Проверьте формат файла (M3U / M3U8)
- Drag & drop поддерживает только указанные расширения

### EPG пустой
- Введите URL XMLTV в настройках (⚙)
- Нажмите кнопку **📡 EPG** — будет первая загрузка
- Каналы привязываются по полю `tvg-id` из M3U
