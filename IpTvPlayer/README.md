# IP TV Player

Современный Windows десктоп-плеер для просмотра IPTV потоков на основе C#/.NET с WPF и LibVLCSharp.

## Возможности

✅ Импорт IPTV плейлистов из файла (.m3u, .m3u8) и по URL  
✅ Парсинг и отображение каналов с логотипами и группировкой  
✅ Встроенный видеоплеер с поддержкой HLS/IPTV потоков  
✅ Поиск и фильтрация каналов по группам  
✅ Управление несколькими плейлистами  
✅ История просмотра каналов  
✅ Сохранение данных в SQLite БД  
✅ Логирование событий приложения  
✅ Темная минималистичная тема интерфейса  

## Требования

- Windows 7 SP1 или выше
- .NET 6.0 Runtime
- VLC библиотека (устанавливается автоматически через NuGet)

## Установка

1. Скачайте последнюю версию приложения
2. Распакуйте архив
3. Запустите `IpTvPlayer.exe`

## Использование

### Импорт плейлиста

1. Нажмите кнопку **"+ Импорт"** на левой панели
2. Выберите файл M3U/M3U8 или введите URL плейлиста
3. Введите название плейлиста
4. Нажмите **"Импортировать"**

### Просмотр каналов

1. Выберите плейлист из списка
2. Канали отобразятся в центральной панели, сгруппированные по категориям
3. Используйте поле поиска для быстрого поиска канала
4. Выберите канал из списка для просмотра

### Управление плеером

- **▶ Проиграть** - начать воспроизведение выбранного канала
- **⏸ Пауза** - приостановить воспроизведение
- **⏹ Стоп** - остановить воспроизведение
- **Громкость** - регулировать уровень звука ползунком

## Решение проблем

### Плеер не воспроизводит видео
- Проверьте, что установлен VLC медиаплеер
- Убедитесь, что URL потока корректен
- Посмотрите логи в папке `Logs/`

### Не загружается плейлист
- Проверьте формат файла (должен быть M3U или M3U8)
- Убедитесь, что файл не повреждён
- Попробуйте переимпортировать плейлист
